# Oscar Castellino Website — Claude Memory

## Project
Static HTML/CSS/JS website for Oscar Castellino, opera baritone.
Hosted on GitHub Pages at oscarcastellino.com
GitHub: https://github.com/goodfornothin/oscarcastellinowebsite

## How to deploy
Make changes, then: git add . && git commit -m 'message' && git push
GitHub Pages auto-deploys within ~2 minutes.

## Key files
- index.html — homepage
- style.css — global styles (dark theme, gold #d4a853 accent)
- data.json — events, biography data
- events/ — individual event landing pages
- pictures/ — artist/performance photos
- videos/ — background videos
